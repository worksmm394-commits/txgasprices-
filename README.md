# txgasprices

Static site that publishes Texas gas prices for 100 cities, deployed to Cloudflare Pages at **[txgasprices.net](https://txgasprices.net)**.

## How it works

1. **`fetch-prices.js`** — pulls the Texas state-average regular-gasoline price from AAA (scraped) and optionally EIA. Writes `prices.json`.
2. **`fetch-apify-prices.js`** — calls the Apify actor `johnvc/fuelprices` (GasBuddy scraper) for the 20 largest Texas cities and records the cheapest price per tracked chain per city. Merges into `prices.json` under `cities.<slug>`.
3. **`generate.js`** — reads `prices.json` + `towns.json` + `texas_gas_site_ui_mockup.html`, emits 500 static city pages (100 cities × 5 fuel variants), a homepage, and `sitemap.xml` to `output/`. Per-city prices from Apify override the estimate; cities without Apify data use the state-average baseline adjusted by regional multiplier.
4. **`build-car-db.js`** — one-shot. Builds `car-db.json` from fueleconomy.gov for the trip-cost calculator's Make/Model/Year picker. Only re-run when the vehicle dataset needs a refresh.

## Local commands

```bash
# Refresh AAA state average
node fetch-prices.js

# Refresh per-city live prices from Apify (needs APIFY_TOKEN env var)
APIFY_TOKEN=apify_... node fetch-apify-prices.js

# Build HTML into output/
node generate.js

# Deploy to Cloudflare Pages
npx wrangler pages deploy output --project-name=txgasprices
```

### Testing a subset with Apify

```bash
APIFY_TOKEN=apify_... CITY_FILTER=houston-tx,dallas-tx node fetch-apify-prices.js
```

## GitHub Actions

Two workflows run automatically — one hourly, one daily.

| File | Schedule | Does |
| --- | --- | --- |
| `.github/workflows/update-prices.yml`       | hourly (UTC `0 * * * *`) | AAA state avg → generate → deploy |
| `.github/workflows/update-apify-prices.yml` | daily (UTC `0 10 * * *`) | Apify per-city → AAA refresh → generate → deploy |

Both deploy to the same Cloudflare Pages project. `fetch-prices.js` preserves the `cities` block written by `fetch-apify-prices.js`, so the hourly workflow can't wipe the daily Apify data.

### Required GitHub repository secrets

In your repo, go to **Settings → Secrets and variables → Actions → New repository secret**, and add:

| Secret name        | Used by               | How to get it |
| ------------------ | --------------------- | ------------- |
| `CF_API_TOKEN`     | both workflows        | [dash.cloudflare.com → My Profile → API Tokens → Create Token → "Edit Cloudflare Workers" template](https://dash.cloudflare.com/profile/api-tokens) |
| `CF_ACCOUNT_ID`    | both workflows        | Cloudflare dashboard → right sidebar of any site or the Workers & Pages section |
| `APIFY_TOKEN`      | daily Apify workflow  | [console.apify.com → Settings → Integrations → Personal API tokens](https://console.apify.com/account/integrations) |
| `EIA_API_KEY`      | optional (fetch-prices) | [eia.gov/opendata/register.php](https://www.eia.gov/opendata/register.php) — free |
| `GASBUDDY_API_KEY` | optional (fetch-prices) | Only if you later add direct GasBuddy API access |

Only `CF_API_TOKEN` + `CF_ACCOUNT_ID` are required for the hourly workflow to run. Add `APIFY_TOKEN` before the daily workflow runs (otherwise it will fail fast).

**Rotate the Apify token** any time it's been exposed in a shell, chat log, commit, or screenshot. Apify runs cost money and a leaked token can be drained.

## Data files

- **`prices.json`** — shared across all three feeds:
  - `stateAverage.*` — from AAA (hourly)
  - `chains[]` — per-chain estimates (AAA + fixed offsets)
  - `cities[slug].regular[]` — per-chain live prices from Apify (daily, top 20 cities only)
  - `cities[slug].cheapestOverall` — cheapest station of any brand in that ZIP
- **`towns.json`** — 100 Texas cities with slug, county, population, lat, lng.
- **`car-db.json`** — 145 makes × 5,407 models × years 1984–2025 for the trip-cost calculator (generated by `build-car-db.js`).

## Output

`generate.js` emits to `output/`:

- 500 `output/gas-prices/<slug>/[<fuel>/]index.html` pages — one per (city × fuel) plus a `/cheapest` alias (noindex'd, canonical to regular)
- `output/index.html` — homepage listing all 100 cities with their current cheapest regular
- `output/sitemap.xml` — 503 URLs
- `output/favicon.ico`, `output/favicon-16x16.png`, `output/favicon-32x32.png`, `output/apple-touch-icon.png`, `output/og-image.png` — static assets (regenerate with `make-favicon.ps1` / `make-og-image.ps1`)

`output/` is in `.gitignore`. CI regenerates it on every deploy.
